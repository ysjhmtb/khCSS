구조 선택자(순번)

특정한 위치(순번)에 있는 태그선택

first-child : 형제 관계 태그 중 첫번째 태그

last-child : 형제 관계 중 마지막 태그

nth-child(순번, 수열) : 형제 관계 태그 중 앞을 기준으로 특정 인덱스 번째 항목을 선택

nth-last-child(순번, 수열) : 형제 관계 태그 중 뒤를 기준으로 특정 인덱스 번째 항목을 선택

형태 구조 선택자

특정한 위치에 있는 태그 선택(태그까지 구분)

:first-of-type : 형제 관계 태그 중 첫번째 태그 선택

:last-of-type : 형제 관계 태그 중 마지막 태그 선택

:nth-of-type : 앞에서 부터 수열(인덱스)번째 태그 선택

:nth-last-of-type : 뒤에서 부터 수열번째 태그 선택

링크 선택자

링크(a -> href)가 걸려있는 태그들을 선택(상태)

:link -> 방문전(링크 홈페이지를 가기전 상태)

:visited -> 방문후(링크 홈페이지를 갔다온 후 상태)

  • 네이버 홈페이지로 이동
  • 구글 홈페이지로 이동
  • 유투브

문자 선택자

태그 내부에 특정 조건의 문자를 선택

선택자(태그)::first-letter : 첫번째 글자 선택

선택자::first-line : 첫번째 줄을 선택

선택자::before : 해당 영역의 앞을 선택

선택자::after : 해당 영역의 뒤를 선택

선택자::selection : 사용자가 드래그한 글자들 선택
