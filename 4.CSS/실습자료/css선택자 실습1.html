CSS 기본 선택자

선택자란?
특정한 HTML 태그를 선택할 때 사용하는 기능으로, 태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

모든 선택자

HTML 문서 안의 모든 태그를 선택할 때 적체 선택자를 사용할 수 있다.

  • 테스트1
  • 테스트2
  • 테스트3

태그 선택자

HTML 문서 내에 같은 태그를 모두 선택할 때 사용한다.

 - p {color : red;} 와 같은 형태로 사용된다.

아이디 선택자

HTML 문서 내에 해당하는 아이디 속성의 값을 가진 태그만 선택할 때 사용한다.

 - #아이디명 {color : blue;} 와 같은 형태로 사용된다.

클래스 선택자

HTML 문서 내에 여러 개의 태그를 동일한 클래스명으로 지정하여 선택할 때 사용한다.

 - .클래스명 {color : purple;} 와 같은 형태로 사용된다.

기본 속성 선택자

태그선택자와 함께 사용, 선택자 뒤에 []를 사용하여 [속성명=값] 형태로 사용

 태그선택자[속성[=속성값]]{ css설정; }



문자 속성 선택자

특정 문자열 값을 확인하여 스타일을 적용하는 방식의 선택자

선택자[속성~=값] : 속성값에 값 포함 여부를 확인하여 해당 노드를 선택하는 선택자

선택자[속성|=값] : 속성 값에 값 포함여부를 확인, -구분하여 값을 인식하여 해당 노드를 선택 함
